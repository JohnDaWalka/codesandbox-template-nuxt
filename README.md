## Features

### WireGuard iOS Configuration Kit

This application includes a custom WireGuard configuration tool optimized for iOS devices. Access it at `/wireguard` to:

- Generate WireGuard VPN configurations
- Customize interface and peer settings
- Export configurations for iOS devices
- View installation instructions for iOS

The WireGuard kit helps you create and manage VPN configurations that work seamlessly with the WireGuard iOS app.
